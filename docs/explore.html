Catalonia liveability

mitma-sedona · geo-browser

Where could you live well in Catalonia — within bike-reach of a train, a climbing gym or yoga nearby, a hospital in reach, and away from heavy industry and motorway noise? This map carries an H3 res-8 multi-criteria liveability index over 45,220 hexagons, built as an applied spatial-data-science pipeline on Apache Sedona from MITMA mobility, OpenStreetMap amenities and environmental layers. Every number is reproducible from the repository.

The liveability score is a relative index — a starting question, not a guarantee, not a recommendation, not an operational service. Dev-scope prototype.

    Scope & caveats

    • Relative index, not a guarantee. The score orders hexes within Catalonia; it makes no absolute or "you should move here" claim.
    • Dev-scope window. The mobility signal is 7 days of March 2024 (2024-03-04 … 10), not a full year — a prototype slice.
    • Sparse amenity coverage. Most of the grid has no nearby station or gym in range; those cells are grey ("none within reach"), which is itself the finding — not a low score.
    • Null is a distinct state. A missing value is rendered slate-grey, never as a ramp-low or zero value.

    How it's scored — dimensions & sources

    The liveability score sums six weighted dimensions over each H3 cell. Each source below is tagged live (wired in this dev-scope prototype) or planned (designed for v2, not yet fed into the gold table). Default weights from configs/weights.yaml.

    DimensionSources
    Mobility & accessibilityw +1.4 Valhalla bike isochrones from train stations train reach (5 km / 3 km buffer approximation) Renfe + FGC GTFS frequency to BCN
    Lifestylew −1.0 OSM sport=climbing (37% of grid) OSM sport=yoga (10% of grid)
    Naturew +0.15…+6 OSM parks / forest / coastline distance Copernicus tree cover WDPA / Natura 2000 iNaturalist density
    Environmental healthw −0.05…−2 EEA + XVPCA + CAMS NO₂ / PM₂.₅ Landsat 8/9 LST — UHI Δ VIIRS DNB light pollution
    Penaltiesw −6 / −12 OSM landuse=industrial density E-PRTR registry motorway within 500 m
    Health amenitiesw −1.0 / +1 OSM hospitals (10% of grid) OSM pharmacy density CatSalut registry
    Mobility "vibe"w 0.0* MITMA daily OD inflow / outflow through-flow ratio

    live = a populated gold column you can recolour the map by right now; planned = the v2 roadmap, currently null (those cells render slate-grey). *the "vibe" weight is 0 by default — set positive in quiet_strict to penalise busy through-flow. Train reach is a buffer approximation pending real Valhalla isochrones.
